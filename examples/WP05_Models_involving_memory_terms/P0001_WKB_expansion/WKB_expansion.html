WKB expansion for a fractional Schr√∂dinger equation

In the particular case under analysis, the construction that we obtain is then applied to the study of controllability properties for the one dimensional fractional Schr√∂dinger equation.
With highly oscillatory initial datum
Nx = 400;
L = 1;
hx = (2*L)/(Nx+1);
Now, we can be define the function_handle u0, from before data.
x0 = 0;
fr = pi^2/16;
gamma = hx^(-0.9);
%
u0 = @(x) exp(-0.5*gamma*(x-x0).^2).*exp(1i*fr*x/hx);
Can see the representation of initial Datum
set(gcf,'Units','normalized','Position',[0.1 0.1 0.6 0.4])
x = -L:hx:L;
%

title('|u_0(x)|')
xlabel('x'); ylabel('u(x)');
%
subplot(1,3,2)
plot(x,real(u0(x)))
title('real(u_0(x))')

%
subplot(1,3,3)
plot(x,imag(u0(x)))
title('img(u_0(x))')
xlabel('x'); ylabel('u(x)');
