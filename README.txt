
DyCon Toolbox es una libreria para el control optimo no lineal que hace enfasis en la sintaxis 
de entrada de los problema. De manera que es capaz de construir problemas de control optimo con muy pocas lineas haciendolo muy legible y practico. Para mas informacion visita:  https://deustotech.github.io/dycon-toolbox-documentation/

## Instalacion

La instalacion debera realizarse mediante el siguiente comando en una consola de MATLAB 9.3 o superior

```
StartDyConToolbox
```
